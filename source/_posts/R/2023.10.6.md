## 取数据框的列名

```R
cols <- colnames(tb)#用于循环遍历
```



## 遍历操作

```R
for (col in cols) {
  #如果列名是变量col,col为一个字符串,用tb[[col]]来获取其值
  #assign是用来字符串变成同名的变量并赋值
  if(all(grepl(pattern,tb[[col]]))){
    assign(paste(col,"_max"),max(myParseInt(tb[[col]])))
  }
}

```
